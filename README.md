# Physical Unclonable Functions Implemented with Analog Behaviour of SRAM
• Designed PUF using SRAM to generate unique device fingerprints for hardware authentication and encryption.


• Examined analog properties such as current and voltage variations in a 6-Transistor based SRAM memory cell.


• Implemented a 4 x 4 16-bit 6-T SRAM memory in LTspice and introduced process variations using Python
